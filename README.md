# Bikkuri

　　『Ｎ！を計算する [ bikkuri.exe ]  Ver.0.70 』

　◎　機　能

　　Ｎの階乗（ただしＮ＜＝１００）を計算する．


　◎　使い方

　　　bikkuri [n]

　　ｎ（０＜ｎ＜＝１００）で指定した整数の階乗を求めます．
　　ｎを指定しない場合，１の階乗から順次計算結果を表示します．


　◎　その他・解説・雑感
